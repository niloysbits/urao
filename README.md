## Overview

`উড়াও` is a playful frontend project built with React, Vite, Tailwind CSS, and Motion.
It turns a normal ride-booking experience into an absurd flying adventure with funny vehicles,
ridiculous payment methods, and fully Bengali humor-driven copy.

## Highlights

- Funny Bengali ride-booking experience with animated UI
- Custom vehicle selection, pilot flow, tracking simulation, and parody payment methods
- Responsive single-page app with a bold visual style
- Built for entertainment, portfolio showcase, and creative frontend experimentation

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Motion
- Lucide React

## Run Locally

**Prerequisites:** Node.js 20+

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
4. Preview the production build:
   ```bash
   npm run preview
   ```

## Available Scripts

- `npm run dev` starts the local development server on port `3000`
- `npm run build` creates the production build
- `npm run preview` previews the built app locally
- `npm run lint` runs TypeScript type-checking
- `npm run clean` removes the `dist` directory
